## 🛠️ Built With

- **Phaser 3**: Game engine for immersive virtual environments.
- **Colyseus**: WebSocket-based server framework for real-time interaction.
- **React/Redux**: Front-end framework to handle dynamic user interfaces.
- **PeerJS**: WebRTC-based library for real-time video and screen sharing.
- **TypeScript & ES6**: For both client-side and server-side development.

## 🌟 Features

- **Proximity-Based Voice Chat**: Engage in conversations with others by getting close to them within the virtual space.
- **Flexible Screen Sharing**: Share your screen with ease during meetings or collaborations.
- **Interactive Whiteboards**: Embed interactive whiteboards to brainstorm, sketch, and collaborate.
- **Customizable Private Rooms**: Create private spaces for team discussions, brainstorming, or socializing.
- **Real-time Text Chat**: Engage in instant messaging with users in the same virtual environment.

## 🏠 Virtual Workspace

- **Proximity Chat**: Voice chat becomes available when close to others, simulating real-life interaction.
- **Flexible & Instant Screen Sharing**: Easily share your screen with others.
- **Text Chat with Dialog Bubbles**: Real-time, interactive chat with smooth bubble effects.
- **Embedded Whiteboards**: Collaborate with others in real-time on digital whiteboards embedded within your space.
- **Custom/Private Rooms**: Personalize your workspace with custom or private rooms for your team or projects.

## 🎮 Controls

- **W, A, S, D** or **arrow keys**: Move around the virtual office.
- **E**: Sit down at a desk or chair.
- **R**: Use a computer (for screen sharing).
- **Enter**: Open the chat window.
- **ESC**: Close the chat window.
